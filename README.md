# SimHackJune2025

Repo of small data to enable local development during the June 2025 hackathon at Duke University.

## Cloning this repository

Before cloning this repository, ou will first need to install Git LFS (Large File System) if not already installed.
Try running
```bash
git lfs install
```

If that is reported as an unknown command, install Git LFS first by following the instructions on their [homepage](https://git-lfs.com/).
Then proceed with cloning as usual
```bash
git clone https://github.com/DukeCosmology/SimHackJune2025
```
